# How to execute this project?
* If you want to run this project. Please, enter the '''$uv run main.py'''

## Configuration for this project
1. Get api key from the upstage dashboard and copy it.
2. create .env file on root directory and make variable, name of UPSTAGE_API_KEY, and paste the api key.

* If you want to use gmail, then please visit this web-site : https://docs.langchain.com/oss/python/integrations/tools/google_gmail
